Overview

Purpose

The purpose of this web application is to provide a place where users can quickly generate simple character sheets for use in old-styled tabletop RPGs, and then store them locally for reference.

Audience

The audience for this app is people who want a quick way to roll up characters for their games without a lot of extra hassle. Rather than creating an app that people have to download, I can just send the link to my friends and they can all access it at once without having to download anything.

Dynamic elements

There will be a "Roll" button that will generate six random ability scores for each stat and then display them on the page by updating the DOM. The user will enter a name, select a class, and their rolled stats will all be stored as an object. This object will be saved in localStorage. A separate page (“The Tavern”) will read stored character data from localStorage and insert character cards into the page using DOM manipulation. The site will use arrays and array methods (such as forEach, map, or filter) to process the scores and information.
